# Frequently Asked Questions

## Q: **Whitch platforms are supported?**

**A:** All major platforms are officially supported, specifically:

- **Desktop**: Windows, Linux, MacOS
- **Mobile**: iOS, Android
- **Universal Windows Platform** (UWP)
- **Web**: WebGL
